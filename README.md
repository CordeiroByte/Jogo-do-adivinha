## CordeiroByte
# Jogo: Descubra o Número 

Um jogo simples desenvolvido em **JavaScript**, onde o objetivo é adivinhar um número aleatório entre 1 e 100 em até 6 tentativas.

---

## Sobre o projeto

Este projeto foi criado para praticar conceitos fundamentais de JavaScript, como:

* Manipulação do DOM
* Eventos (`submit`, `click`)
* Estruturas condicionais
* Arrays e métodos (`push`, `includes`, `sort`)

---

## Como funciona

* O sistema sorteia um número aleatório entre **1 e 100**
* O jogador tem **6 tentativas** para acertar
* A cada erro, o jogo fornece uma **dica inteligente**, indicando um intervalo possível
* O jogo impede números repetidos
* Ao acertar ou esgotar as tentativas, o jogo é finalizado

---

## Exemplo de dica

```text
Dica: é maior que 30 e menor que 50
```
